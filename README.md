# My ~/.emacs.d Setup

This is a safe place for me to keep my .emacs.d folder and .emacs file, so I don’t need to keep trying to use Dropbox or copy/paste it everywhere.

## Setup

Just `git clone` this repository from your home directory, `cd` into it, and then run `setup.cmd` (Windows) or `setup.sh` (Linux/Mac) and it will install all the necessary plugins and put the `.emacs` file where it needs to go.
